# Catalogo de videojuegos

Catalogo de videojuegos es una aplicación web desarrollada con React y TypeScript que permite explorar videojuegos utilizando la API de RAWG. Los usuarios pueden buscar títulos, filtrarlos por género y guardar sus juegos favoritos para consultarlos posteriormente.

- Explorar catalogo de juegos
- Buscar nombres por titulo
- Filtrar juegos por genero
- Agregar y eliminar juego a favoritos
- Almacenar favoritos en localstorage

## Instalación

1. Clona el repositorio.

```bash
git clone https://github.com/DiegoBeja/Catalogo_de_Videojuegos.git
```

2. Entra al directorio del proyecto.

```bash
cd catalogo_de_videojuegos
```

3. Instala las dependencias.

```bash
npm install
```

4. Inicia el servidor de desarrollo.

```bash
npm run dev
```

## API

Este proyecto utiliza la **RAWG Video Games Database API**, la cual proporciona información sobre miles de videojuegos, incluyendo su nombre, imagen, fecha de lanzamiento, calificación, plataformas y géneros.

Más información: https://rawg.io/apidocs
